# LED-Blink-At-Random
An LED that blinks at random interval using Arduino 

# Component used 
1. Arduino UNO
2. Resistor
3. Wires

# How to Run
1. Open the `sketch_mar27random.ino` file in the Arduino IDE.
2. Connect your Arduino to your computer.
3. Select your Board and Port from the Tools menu.
4. Click **Upload**.
